# Конвертер групп вершин (клики) в список смежности (hyperedge_to_adjlist.cpp)

## Описание

Программа конвертирует группы вершин в граф, представляя каждую группу как полный подграф (клика).
На выходе программа записывает **список смежности** в файл.

Проект демонстрирует:
- C++ (STL: `vector`, `sort`, удаление дубликатов)
- чтение/запись файлов (`ifstream`/`ofstream`)
- обработку ошибок
- работу с графами (клики, смежность)

---

## Формат входных данных

1. Файл `n.txt` содержит одно число `n` — количество файлов с группами вершин.
2. Для каждого файла `i.txt` (`i = 1..n`) формат:
	- `count_top` — количество вершин в группе  
	- `v1, v2, ..., v_count_top` — индексы вершин (нумерация с 1)

# Алгоритм Прима — минимальное остовное дерево (prim_algorithm.cpp)

## Описание

Программа строит **минимальное остовное дерево** для **взвешенного неориентированного графа** с помощью **алгоритма Прима**.
Алгоритм начинает с выбранного пользователем вершины и добавляет в дерево ребро минимального веса, которое соединяет посещенные вершины с еще непосещенными.
Если граф несвязный, выводится сообщение.

Проект демонстрирует:
- C++ (STL: `vector`, `pair`, `priority_queue`)
- работа со взвешенными графами (список смежности)
- реализация алгоритма Прима (MST)
- чтение/запись файлов (`ifstream`/`ofstream`)
- базовая проверка корректности результата (проверка связности)

---

## Формат входных данных

Файл `input.txt` содежит:
- count_top count_edge
- u1 v1 w1
- u2 v2 w2
где:
`count_top` - количество вершин,
`count_edge` - количество ребер,
`ui vi` - вершины ребра (нумерация с 1)
`wi` - вес ребра

Ребро добавляетя в обе стороны, так как граф **неориентированный**.

## Формат выходных данных

Результат записывается в файл `output.txt` в виде списка ребер с весом.
