# Метод Нелдера-Мида - оптимизация функции двух переменных (nelder_mead.cpp)

## Описание
Программа реализует **метод Нелдера-Мида** - алгоритм **безградиентной оптимизации** (поиск экстремума) функции двух переменных.

Основные операции в реализации метода:
- **reflection** (отражение)
- **expansion** (растяжение)
- **contraction** (сжатие)
- **shrink** (редукция)

---

## Формат входных данных
Ввод через консоль двух координат начальной точки (x, y)
Оптимизируемая функция задается в функции Calc_f().
Основыне параметры алгоритма:
- `max_iter` - максимальное число итерацийб
- `eps` - точность остановки (критерий сходимости)

## Формат выходных данных
Результаты записываются в файл `result.txt` в формате: x y f(x,y).
Сохраняется лучшая вершина симплекса на каждой итерации.
