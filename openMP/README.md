# Демонстрация OpenMP (параллельные секции и распределение циклов) (openMP_demo.cpp)

## Описание

Проект демонстрирует основы параллельного программирования на C++ с использованием OpenMP:
- получение количества доступных потоков
- параллельная секция `#pragma omp parallel`
- распараллеливание цикла `#pragma omp for`
- сравнение стратегий распределения итераций:
    - без явного указания schedule
    - `schedule(static, chunk)`
    - `schedule(dynamic, chunk)`

Проект демонстрирует:
- C++ консольное приложение
- OpenMP (`omp_get_max_threads`, `omp_get_thread_num`, `omp_get_num_threads`)
- параллельные секции
- планирование циклов (`static`, `dynamic`)
- базовые навыки отладки параллельного выполнения

# Вычислительные задачи с использованием OpenMP (openMP_tasks.cpp)

## Описание

Проект представляет собой набор учебных задач по параллельному программированию и численным вычислениям на С++ с использованием OpenMP.

В програме реализованы:
1) скалярное произведение двух векторов
2) умножение матриц и вычисление нормы результата
3) работа с верхнетреугольной матрицей (обратный ход), вычисление верктора y
4) линейная комбинация векторов и вычисление

## Формат выходных данных
Вывод в консоль: время выполнения заданий и нормы результатов. Файл `XandY.txt`: `x[i]` и `y[i]` для 3 задания.
