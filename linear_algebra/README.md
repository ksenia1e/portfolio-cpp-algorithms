# Решение СЛАУ методом LDL^t - разложения (профильный формат матрицы)

## Описание
Данный проект реализует решение системы линейных алгебраический уравнений (СЛАУ) A x = b с помощью **LDL^t - разложения** для **симметричной**, заданной в **профильном формате**.
Алгорим ориентирован на экономию памяти и работу с разреженными матрицами без явного хранения всей матрицы в плотном виде.

---

## Что реализовано
- хранение симметричной матрицы в профильном формате:
    - `di[]` - диагональные элементы
    - `al[]` - элементы нижнего треугольника в профильном виде
    - `ia[]` - указатели начала профиля строк
- разложение:
\[
A = L D L^T
\]
где:
- L - нижнетреугольная матрица
- D - диагональная матрица знаков
- L^t - транспонированная L

- решение СЛАУ:
    1) прямой ход Ly = b
    2) диагональное преобразование Dz = y
    3) обратный ход L^tx = z

---

## Формат входных данных

Программа читает данные из файлов:

- `n.txt` — размерность матрицы `n`
- `ia.txt` — массив указателей профиля строк (`n+1`)
- `di.txt` — диагональ матрицы (`n`)
- `al.txt` — элементы профиля нижнего треугольника
- `b.txt` — вектор правой части (`n`)

> В программе предусмотрена генерация матрицы типа Гильберта `Gilbert()`, которая формирует файлы `n.txt`, `ia.txt`, `di.txt`, `al.txt`.

---

## Выходные данные

Программа выводит в консоль:

- массив `al[]` (профиль нижнего треугольника)
- массив `di[]` (после разложения хранит диагональные элементы разложения)
- массив `ia[]`
- массив `di_signed[]` (знаки диагонали)
- найденное решение `x[]`
