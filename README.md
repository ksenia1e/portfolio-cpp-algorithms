# Портфолио проектов на C++ (алгоритмы, численные методы, параллельные вычисления)

## Структура портфолио
- `graphs/` - алгоритмы работы с графами  
  - `hyperedge_to_adjlist.cpp` - преобразование гиперрёбер в список смежности  
  - `prim_algorithm.cpp` - алгоритм Прима  

- `openMP/` - проекты с параллельными вычислениями  
  - `openMP_demo.cpp` - демонстрация `#pragma omp parallel` и `#pragma omp for`  
  - `openMP_tasks.cpp` - задачи на распараллеливание  

- `linear_algebra/` - численные методы и линейная алгебра  
  - `ldlt.cpp` - решение СЛАУ методом LDL^t-разложения  

- `optimization/` - численные методы и оптимизация  
  - `neld_mied.cpp` - метод Нелдера-Мида для поиска экстремума
